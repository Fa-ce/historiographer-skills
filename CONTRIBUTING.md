# Contributing

## 提交一个新 skill

1. 在 `skills/<skill-name>/` 下创建目录。
2. 添加 `SKILL.md`，写清楚用途、触发场景、输入输出和限制。
3. 如有需要，可增加以下子目录：
   - `scripts/`
   - `references/`
   - `templates/`
   - `assets/`
4. 保持目录自包含，不引入无关文件。
5. 提交 Pull Request。

## 命名规范

- 目录名使用 `kebab-case`
- 文件名保持简洁明确

## 审核标准

- 目标明确，适用场景清楚
- 不包含无关内容
- 说明可以被其他人直接复用
